
# Setting up the Application

1. Unzip the app
2. Install Node.js >=v14.0 
- See installation [guide](https://nodejs.org/en/download/)
3. Install Yarn 
- `npm install --global yarn`
4. Install NPX
- `npm install -g npx`
5. Install Node Module Dependencies
- `yarn`


# Running the Project
`npm start`
